# MyPeople

Note: Do setup and run from the project directory  

## Setup:
`npm install -g nodemon`  
`npm install`  

## Run:
`npm run mongod` -> Creates datbase path at `data/database` and starts mongo    
`npm start` -> runs `nodemon app`
